# Distributed-AI-Systems

This is the source code for book *Distributed AI Systems* by Fuheng Wu.

![](https://m.media-amazon.com/images/I/41hrIqYTxSL.jpg)

For chapter 1-4, to run the code, install the dependencies:

```bash
pip install torch torchvision numpy distai
```

For other chapters, please check the requirements.txt in each chapter folder, and run:

```bash
pip install -r requirements.txt
```
